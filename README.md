# 🔍 How It Works

1. Upload company documents
2. Extract text from files
3. OCR scanned PDFs if needed
4. Split text into chunks
5. Generate vector embeddings
6. Store embeddings in ChromaDB
7. Perform semantic retrieval
8. Generate AI responses locally
